N x N

그리드 미디어 콜라주 → 영상

App Store 심사 대기중

N x N은 사진과 영상을 그리드로 배치하고 한 편의 영상으로 내보내는 앱입니다. 1x1, 2x2, 3x3 레이아웃 중 선택해 셀마다 미디어와 이모지 타이틀을 채우면, 1080×1350 H.264 영상으로 변환됩니다. 모든 작업은 기기에서 로컬로 처리됩니다.

주요 기능

유연한 그리드 레이아웃

1x1, 2x2, 3x3 중 선택. 레이아웃을 바꿔도 미디어는 보존됩니다.

사진 + 영상 + 이모지

셀마다 사진/영상을 채우고 이모지로 타이틀을 더하세요.

1080×1350 영상 출력

H.264, 30fps, 4:5 비율로 어디서든 공유하기 좋은 영상.

로컬 처리, 프라이버시 우선

모든 미디어는 기기에서 처리되며, 서버에 저장되지 않습니다.

곧 만나요

App Store 심사가 완료되는 대로 다운로드 링크가 공개됩니다.
