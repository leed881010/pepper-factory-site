N x N

영상 편집자가 아닌 사람들을 위한 영상 만들기 도구.

Reels 와 Stories 가 세상을 지배하지만, CapCut 이나 iMovie 같은 도구들은 여전히 평범한 사람들에게 트랙과 키프레임으로 사고하라고 요구합니다. 대부분은 한 번 열어보고, 길을 잃고, 조용히 포기합니다. 회고 영상은 끝내 만들어지지 않습니다.

N x N 은 "영상을 편집한다"를 "보드를 채운다"로 바꿉니다. 레이아웃을 고르고, 셀마다 사진이나 영상을 넣고, 폴라로이드처럼 이모지 캡션을 답니다. 추출 버튼 한 번. Reels 와 Stories 용 깔끔한 영상이 나옵니다 — 타임라인, 트랙, 튜토리얼, 결제 장벽 없음.

보드 위에서 할 수 있는 것

6가지 보드 레이아웃

1×1, 1×2, 1×3, 2×2, 3×3, 9:16 풀스크린. 레이아웃을 바꿔도 미디어는 보존됩니다.

폴라로이드 캡션

각 셀 위에 이모지 타이틀. 폴라로이드 다이어리처럼 보드 자체가 구도를 잡아줍니다.

자유 배치 텍스트

커스텀 폰트, 외곽선, 색상 팔레트. 인스타 수준의 텍스트 렌더링.

원터치 스티커

온디바이스 Vision Subject Lifting 으로 사진의 피사체를 잘라 스티커로. 여러 피사체면 선택 시트.

클립보드 → 스티커

어떤 이미지든 복사해서 붙여넣으면 즉시 스티커로 변환. 외부 소스도 자유롭게.

로컬 처리, 프라이버시 우선

모든 미디어는 기기에서 처리됩니다. 서버 업로드 없음, 결제 장벽 없음.
